Important textbook line

১ হাজার থেকে ৩ হাজার সেলুলোজ অণু মিলে ১টি সেলুলোজ চেইন গঠিত হয়। প্রায় ১০০টি সেলুলোজ চেইন মিলে ১টি ক্রিস্টালাইন মাইসেলি, ২০টি মাইসেলি মিলে ১টি মাইক্রোফাইব্রিল, ২৫০টি মাইক্রোফাইব্রিল মিলে ১টি ম্যাক্রোফাইব্রিল, আর অনেক ম্যাক্রোফাইব্রিল মিলে ফাইবার তৈরি করে।

Shoron পুরো paragraph থেকে exam-useful sequence আলাদা করে, তারপর সেটাকে quick visual memory pattern-এ ভেঙে দেয়.
Shoron mnemonic

সেলু চাচা মাইসাকে
মাইক্রো-ম্যাক্রোতে
ফাইবার করলো :
১–৩K, সেঞ্চুরি, T20,
আড়াইশো, অনেক

1
(সেলু চাচা) + (১–৩K)
সেলুলোজ চেইন। ১–৩ হাজার সেলুলোজ অণু মিলে ১টি সেলুলোজ চেইন তৈরি করে।
2
(মাইসা) + (সেঞ্চুরি)
ক্রিস্টালাইন মাইসেলি। ১০০ সেলুলোজ চেইন মিলে ১টি ক্রিস্টালাইন মাইসেলি তৈরি করে।
3
(মাইক্রো) + (T20)
মাইক্রোফাইব্রিল। ২০টি ক্রিস্টালাইন মাইসেলি মিলে ১টি মাইক্রোফাইব্রিল তৈরি করে।
4
(ম্যাক্রোতে) + (আড়াইশো)
ম্যাক্রোফাইব্রিল। ২৫০টি মাইক্রোফাইব্রিল মিলে ১টি ম্যাক্রোফাইব্রিল তৈরি করে।
5
(ফাইবার) + (অনেক)
অনেকগুলো ম্যাক্রোফাইব্রিল মিলে ফাইবার বা তন্তু গঠন করে।
